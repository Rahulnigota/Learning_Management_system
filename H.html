HTML

HTML, which stands for HyperText Markup Language, is the standard markup language used to create and design web pages. It is a fundamental building block of the World Wide Web and is used to structure content on the internet.

CSS

CSS, which stands for Cascading Style Sheets, is a style sheet language used for describing the presentation of a document written in HTML or XML.CSS is used to control the layout and appearance of web pages.By using CSS, web developers can define styles such as colors, fonts, spacing, and positioning for HTML elements, creating a visually appealing and consistent look for a website.

JavaScript

JavaScript is a high-level, dynamic, interpreted programming language that is primarily known for its use in web development. It enables developers to add interactive elements to websites, creating dynamic and engaging user experiences. JavaScript is an essential technology for front-end web development and is supported by all modern web browsers.
